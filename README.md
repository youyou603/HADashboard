## üîß Troubleshooting

If you encounter any problems while installing or using HADashboard, consider the following steps:

- **Installation Issues:** Make sure you have allowed apps from unknown sources in your settings. Check your device storage for space.
- **Connectivity Problems:** Ensure your tablet is connected to the same Wi-Fi network as your Home Assistant instance.
- **Feature Not Working:** Restart both the HADashboard app and the Home Assistant server if features do not respond as expected.
